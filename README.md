# Choreflow API

## Running the API

### Prerequisites

- Go 1.22.1

### Install Dependencies

```bash
go mod tidy
```

## Documentation

- **Base URL:** `http://localhost:3211/choreflow/api/v1`
- **Full URI:** `http://localhost:3211/choreflow/api/v1/{endpoint}`
- **Swagger UI**: `http://localhost:3211/choreflow/api/v1/swagger/ui`
